Features

Searching

You can search for a contact using by typing into the search bar. It filters by name.

Sorting

You can sort your contacts in alphabetical ascending or descending order using the sort button next to the filter field.

Responsive Design

This site should display properly on all standard devices. The contact details display switches to modal mode when the screen's width gets too small. Try it.

Auto-Generating Default Profile Pictures

You might notice that the contacts have colourful backgrounds for their default profile pictures - these are not random. They are generated based on the contact's name. The colours are based on the alphbetical characters present in their name. Here's the general idea:

1. The names are split into three roughly equal groups.

2. Each character is given a value of 0-25, based on its position in the alphabet. (a = 0, z = 25)

3. The average alphabetical position of the letters of each group is calculated.

4. This average is converted to a percentage of the way throug the alphabet. (a = 0, z = 1)

5. This percentage is mapped to a range of 0-255.

6. These three numbers can then be used as rgb values.

Theoretically similar names should have similar colours overall.
